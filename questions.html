Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: We will get the text with HTML content. In innerHTML, We can read and set HTML content inside a element.
innerText: We will get only visible text.
textContent: We will get only text.

Question-5: How do you create, add, and remove elements using JavaScript?

To create elements, First of all we will set a Id where we want to add an element. for example the id is ('element'). after creating element then we have to set any innerText

const elements = document.getElementById ('element') const createP = document.createElement('p'). createP.innerText = "I am a web developer"

To add Elements, first of all we have to select all the classlist then we can add.
const elements = document.getElementById('element'); elements.classlist.add("bangladesh")

To remove elements, first of all we have to select all the classlist then we can remove
const elements = document.getElementById('element'); elements.classlist.remove("bangladesh")
